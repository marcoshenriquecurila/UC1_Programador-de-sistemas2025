O que é CSS e o que ele faz?

Introdução ao CSS

CSS, que significa Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML. Enquanto o HTML define a estrutura e o conteúdo da página, o CSS é responsável por definir o layout, as cores, as fontes e outros aspectos visuais.

Como o CSS funciona?

O CSS permite separar a estrutura da página do estilo visual, o que facilita a manutenção e a atualização do design. Ele pode ser aplicado de três maneiras:

  • Inline: Aplique o CSS diretamente no elemento HTML, utilizando o atributo style.
  • Interno: Insira o CSS dentro da tag <style> no cabeçalho da página HTML.
  • Externo: Crie um arquivo CSS separado e vincule-o à página HTML usando a tag <link>.

Principais Funcionalidades do CSS

O CSS oferece uma ampla gama de funcionalidades para estilizar e controlar o design de uma página. Algumas das principais funcionalidades incluem:

  • Alteração de cores: Você pode mudar a cor de fundo, textos, bordas e outros elementos.
  • Definir fontes: O CSS permite escolher fontes específicas, tamanhos e estilos de texto.
  • Layouts: Controle de layout de páginas, como o uso de grids, flexbox e posicionamento absoluto ou relativo.
  • Responsividade: Criação de designs que se adaptam a diferentes tamanhos de tela, como para dispositivos móveis, tablets e desktops.
  • Transições e animações: O CSS também pode ser usado para adicionar animações e transições suaves aos elementos da página.

Exemplo Básico de CSS

Aqui está um exemplo simples de como usar o CSS para estilizar uma página HTML:

<style>
    body {
        background-color: #f0f0f0;
        font-family: Arial, sans-serif;
    }
    h1 {
        color: #007BFF;
    }
    p {
        color: #555;
    }
</style>

Este código irá alterar a cor de fundo da página para um cinza claro, definir a fonte padrão como Arial e mudar a cor do título (h1) para azul.

Conclusão

O CSS é uma linguagem essencial para criar páginas web visualmente agradáveis e responsivas. Com ele, os desenvolvedores podem controlar a aparência e o layout das páginas, melhorando a experiência do usuário. Aprender CSS é fundamental para quem deseja criar sites modernos e atrativos.
